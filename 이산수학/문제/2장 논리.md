06. ( 조건명제 ) p -> q 에서 p를 q의 ( 충분조건 ) 이라 하고, q를 p의 ( 필요조건 ) 이라 한다.

* P는 Q를 충족시키기 충분하다는 뜻에서 충분조건이 된다.
